# Display Formatting Options

## Overview
Defines the various formatting options available for customizing the output display.

## Format Types

### List Formats
- Single column (-1)
- Multiple columns (-C)
- Long format (-l)
- Comma-separated (-m)
- With file size (-s)
- One per line (-x)

### Information Selection
- Show inode (-i)
- Show block size (-s)
- Human readable sizes (-h)
- Show entry type (-F)
- Hide group information (-g)
- Show numeric UID/GID (-n)

## Visual Formatting

### Color Support
- Enable/disable colors (--color)
- Color configuration
- File type colors
- Permission-based colors

### Size Formatting
- Binary prefixes
- SI prefixes
- Block size units
- Floating point precision
