# Entry Filtering Options

## Overview
Defines the various filtering options available for controlling which entries are displayed.

## Filter Types

### Pattern Matching
- Include patterns
- Exclude patterns
- Regular expressions
- Case sensitivity

### Entry Type Filters
- Show hidden files (-a, -A)
- Directory only (-d)
- File type selection
- Symbolic link handling

### Time Filters
- Modified time
- Access time
- Creation time
- Time range selection

## Special Handling

### Ignore Patterns
- Backup file ignoring
- Pattern-based exclusion
- Directory exclusion
- Special file handling

### Sort Selection
- Time-based sorting
- Size-based sorting
- Extension sorting
- Version sorting
